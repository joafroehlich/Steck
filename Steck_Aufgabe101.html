Der Graph einer achsensymmetrischen ganzrationalen 
Funktion 4. Grades hat Nullstellen bei -5 und 3 
und geht durch den Punkt (0|1). 
Wie lautet seine Funktionsgleichung? 



Allgemeine Form einer achsensymmetrischen 
ganzrationalen Funktion 4. Grades: 
 
f(x) = ax4 + cx2 + e 
 
f’(x) = 4ax3 + 2cx  
 
f’’(x) = 12ax2 + 2c  

3 Bedingungen: 

1. Hat eine Nullstelle bei -5 bedeutet: 
  
f(-5) = 0 --> 

a * (-5)4 + c * (-5)2 + e = 0 --> 

625a + 25c + e = 0      I 
 
2. Hat eine Nullstelle bei 3 bedeutet: 
  
f(3) = 0 --> 

a * 34 + c * 32 + e = 0 --> 

81a + 9c + e = 0      II 
 
3. Geht durch den Punkt (0|1) bedeutet: 
 
f(0) = 1 --> 

a * 04 + c * 02 + e = 1 --> 

e = 1 
 
I * 9 + II * (-25) (e = 8 eingesetzt)

-2025a - 225c - 25 = 0

a = 1/225 in II eingesetzt. 
 
81 * (1/225) + 9c + 1 = 0  
 
81/225 + 9c + 225/225 = 0 
 
         3067225 + 9c = 0 |-306/225 

                   9c = -306/225 |:9 
 
                    c = -34/225   
 
Gesuchte Funktionsgleichung: 
 
f(x) = (1/225)x4 - (34/225)x2 + 1
