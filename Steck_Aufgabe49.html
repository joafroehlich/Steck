Allgemeine Form einer ganzrationalen Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d

f’’(x) = 6ax + 2b

-a + b - c + d = 0      I 
 
2. Schneidet die y-Achse an der Stelle y = -4 bedeutet: 
 
f(0) = - 4 --> 

a * 03 + b * 02 + c * 0 + d = -4 --> 

d = -4

(d = -4 eingesetzt):

a + b + c = 2     II

6a + 2b = 0       III
