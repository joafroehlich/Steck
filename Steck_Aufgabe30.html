hat bei x = 2/3 einen Wendepunkt, schneidet an der 
Stelle x = -2 die x-Achse, und die Normale 
f(x) = (5/3)x - 5/3 schneidet ihn an der Stelle x = 1. 
Wie lautet seine Funktionsgleichung?



Allgemeine Form einer ganzrationalen Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b 

4 Bedingungen: 

1. Hat bei x = 2/3 einen Wendepunkt bedeutet: 
  
f’’(2/3) = 0 -->

f(-2) = 0 --> 

a * (-2)3 + b * (-2)2 + c * (-2) + d = 0 -->  

-8a + 4b - 2c + d = 0        II 
 
3. Die Normale f(x) = (5/3)x - 5/3 schneidet ihn an 
der Stelle x = 1 bedeutet: 
 
f(1) = 5/3 * 1 - 5/3 = 0 --> 
 
Der Punkt (1|0) liegt auf dem Graphen: 
 
f(1) = 0 -->

a + b + c + d = 0     III 
 
4. Die Normale hat eine Steigung von 5/3.
