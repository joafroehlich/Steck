Der Graph einer achsensymmetrischen ganzrationalen 
Funktion 4. Grades hat im Punkt (2|0) die 
Steigung 2 und bei x = -1 einen Wendepunkt. 
Wie lautet seine Funktionsgleichung?



Allgemeine Form einer achsensymmetrischen 
ganzrationalen Funktion 4. Grades: 
 
f(x) = ax4 + cx2 + e 
 
f’(x) = 4ax3 + 2cx  
 
f’’(x) = 12ax2 + 2c  

3 Bedingungen: 

1. Hat im Punkt (2|0) die Steigung 2 bedeutet zum 
einen: 
 
f(2) = 0 --> 

a * 24 + c * 22 + e = 0 --> 

16a + 4c + e = 0          I 
 
2. Hat im Punkt (2|0) die Steigung 2 bedeutet zum 
anderen: 
 
f’(2) = 2 --> 

4a * 23 + 2c * 2 = 2 --> 

32a + 4c = 2          II 
 
3. Hat bei x = -1 einen Wendepunkt bedeutet:  
 
f’’(-1) = 0 --> 

12a * (-1)² + 2c = 0 --> 

12a + 2c = 0      III   
 
III * (-2) + II 

-24a - 4c = 0 
 32a + 4c = 2

a = 0,25 in III eingesetzt: 
 
12 * 0,25 + 2c = 0 
 
        3 + 2c = 0 |-3 
 
            2c = -3 |:2 
 
             c = -1,5    
 
a = 0,25 und c = -1,5 in I eingesetzt:
