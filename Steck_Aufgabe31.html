geht durch den Nullpunkt, hat bei (1|1) ein 
Maximum und einen Wendepunkt bei (3|f(3)). 
Wie lautet seine Funktionsgleichung? 



Allgemeine Form einer ganzrationalen 
Funktion 3. Grades:

f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b

anderen:

3a * 12 + 2b * 1 + c = 0 --> 

3a + 2b + c = 0        II
