Der Graph einer ganzrationalen Funktion 2. Grades 
geht durch die Punkte (-1|-3), (1|1) und (-2|1). 
Wie lautet seine Funktionsgleichung?


 
Allgemeine Form einer ganzrationalen 
Funktion 2. Grades: 
  
f(x) = ax² + bx + c 
 
f’(x) = 2ax + b 
 
f’’(x) = 2a 
 
3 Bedingungen: 

1. Geht durch den Punkt (-1|-3) bedeutet: 
 
f(-1) = -3 --> 

a * (-1)² + b * (-1) + c = 2 --> 

a - b + c = - 3         I 
 
2. Geht durch den Punkt (1|1) bedeutet:  
 
f(1) = 1 --> 

a * 1² + b * 1 + c = 3 --> 

a + b + c = 1                    II         
 
3. Geht durch den Punkt (-2|1) bedeutet:  
 
f(-2) = 1 --> 

a * (-2)² + b * (-2) + c = 1 --> 

4a - 2b + c = 1        III
