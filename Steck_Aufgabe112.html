Aufgabe 112

Der Graph einer zu (0|0) punktsymmetrischen 
ganzrationalen Funktion 5. Grades hat in (0|0) 
die Tangente y = 7x und in (1|0) einen 
Wendepunkt. Wie lautet seine Funktionsgleichung?



Allgemeine Form einer zu (0|0) punktsymmetrischen 
ganzrationalen Funktion 5. Grades: 
 
f(x) = ax5 + cx3 + ex 
 
f(x) = 5ax4 + 3cx4 + e 
 
f’’(x) = 20ax3 + 6cx 

4 Bedingungen : (eine mehr als nötig) 
 
1. Geht durch (0|0) bedeutet: 
 
f(0) = 0 --> 

a * 05 + c * 03 + e * 0 = 0 --> 

0 = 0 liefert kein Ergebnis   
 
2. Hat in (0|0) die Tangente y = 7x bedeutet:
(Steigung = 7) 
 
f’(0) = 7 --> 

5a * 04 + 3c * 02 + e = 7 --> 

e = 7          
 
3. Hat in (1|0) einen Wendepunkt bedeutet zum 
einen: (e = 7 eingesetzt) 
 
f(1) = 0 --> 

a * 15 + c * 13 + 7 = 0 --> 

a + c + 7 = 0         I   
 
4. Hat in (1|0) einen Wendepunkt bedeutet zum 
anderen: 
 
f’’(1) = 0 --> 

20a * 13 + 6c * 1 = 0 --> 

20a + 6c = 0          II  
 
I * (-6) + II 

-6a - 6c - 42 = 0 
20a + 6c      = 0 
-------------------
14a       - 42 = 0 |+42 
 
           14a = 42 |:14 
 
             a = 3 
 
a = 3 in I eingesetzt. 
 
3 + c + 7 = 0 
 
   10 + c = 0 |-10 

        c = -10 

Gesuchte Funktionsgleichung: 

f(x) = 3x5 - 10x3 + 7x
