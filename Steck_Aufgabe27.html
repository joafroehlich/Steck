Der Graph einer ganzrationalen Funktion 3. Grades 
tangiert an der Stelle x = 0 die x-Achse und hat 
in (2/3|-16/27) einen Wendepunkt. 
Wie lautet seine Funktionsgleichung?



Allgemeine Form einer ganzrationalen 
Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b 

4 Bedingungen: 

1. Tangiert an der Stelle x = 0 die x-Achse 
bedeutet zum einen: 
 
f(0) = 0 --> 

a * 03 + b * 02 + c * 0 + d = 0 --> 

d = 0      
 
2. Tangiert an der Stelle x = 0 die x-Achse 
bedeutet zum anderen: 
 
f’(0) = 0 --> 

3a * 03 + 2b * 02 + c = 0 --> 

c = 0           
 
3. Hat in (2/3|- 16/27) einen Wendepunkt bedeutet 
zum einen: (c = 0 und d = 0 eingesetzt): 
 
f(2/3) = 0 --> 

a * (2/3)3 + b * (2/3)2 = -16/27 -->  
 
(8/27)a + (4/9)b = -16/27        
 
(4/9)((2/3)a + b) = -16/27 |:4/9 
 
(2/3)a + b = -4/3       I 
 
4. Hat in (2/3|- 16/27) einen Wendepunkt bedeutet 
zum anderen: 
 
f’’(2/3) = 6a * (2/3) + 2b = 0 --> 

4a + 2b = 0     II 
 
I * (-2) + II 
 
(-4/3)a - 2b  = 8/3 
     4a + 2b  = 0 
-------------------     
 (8/3)a = 8/3 |:8/3 
 
      a = 1  
 
a = 1 in II eingesetzt: 

4 + 2b = 2 |-4 

2b = - 4 |:2 

b = 2 

Gesuchte Funktionsgleichung: 

f(x) = x3 - 2x2
