Der Graph einer ganzrationalen Funktion 3. Grades 
berührt an der Stelle x = 1 die x-Achse und hat in 
(3|-16) einen Wendepunkt. 
Wie lautet seine Funktionsgleichung? 


 
Allgemeine Form einer ganzrationalen Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b 
 
4 Bedingungen: 

1. Berührt an der Stelle x = 1 die x-Achse bedeutet zum einen: 

f(1) = 0 --> 

a * 13 + b * 12 + c * 1 + d = 4 --> 

a + b + c + d = 0   I 

2. Berührt an der Stelle x = 1 die x-Achse bedeutet zum anderen: 

f’(1) = 0 --> 

3a * 12 + 2b * 1 + c = 0 --> 

3a + 2b + c = 0     II 

3, Hat in ((3|-16) einen Wendepunkt bedeutet zum einen: 

f(3) = -16 --> 

a * 33 + b * 32 + c * 3 + d = 0 -->  

27a + 9b + 3c + d = - 16       III
