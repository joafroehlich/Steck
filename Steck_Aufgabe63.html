Aufgabe 63

Der Graph einer ganzrationalen Funktion 3. Grades 
berührt im Koordinatenursprung die x-Achse und hat 
einen Hochpunkt bei (2|2). 
Wo liegen seine Nullstellen? 



Allgemeine Form einer ganzrationalen 
Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b 

4 Bedingungen: 

1. Berührt im Koordinatenursprung die x-Achse 
bedeutet zum einen: 
  
f(0) = 0 --> 

a * 03 + b * 02 + c * 0 + d = 0 --> 

d = 0 
 
2. Berührt im Koordinatenursprung die x-Achse 
bedeutet zum anderen: 
  
f’(0) = 0 --> 

3a * 02 + 2b * 0 + c = 0 --> 

c = 0 
 
3. Hat einen Hochpunkt bei (2|2) bedeutet zum 
einen (c = 0 und d = 0 eingesetzt): 
  
f(2) = 2 --> 

a * 23 + b * 22 = 0 --> 

8a + 4b = 2       I 
 
4. Hat einen Hochpunkt bei (2|2) bedeutet zum 
anderen (c = 0  eingesetzt): 
  
f’(2) = 0 --> 

3a * 22 + 2b * 2 = 0 --> 

12a + 4b = 0        II 
 
I * (-1) + II  

-8a - 4b = -2 
12a + 4b =  0 
--------------
 4a      = -2  |:4 
 
       a = -0,5     
 
a = -0,5 in II eingesetzt: 
 
12 * (-0,5) + 4b = 0 
 
         -6 + 4b = 0 |+6 
 
              4b = 6 |:4 
 
               b = 1,5 
 
Funktionsgleichung: 

f(x) = -0,5x3 + 1,5x2

Berechnung der Nullstellen: 
 
Es gilt: f(x) = 0 

-0,5x³ + 1,5x² = 0 

-0,5x² * (x - 3) = 0 
 
Ein Produkt aus 2 Faktoren ist dann gleich 0, 
wenn einer der Faktoern gleich Null ist. 
 
1. Faktor: 

-0,5x² = 0 |:(-0,5) 
 
x1,2 = 0 
 
2. Faktor: 
 
x - 3 = 0  |+3 
 
x3 = 3 

Die Nullstellen liegen bei: 

N1,2 = (0|0)     Berührpunkt 

und 

N3 = (3|0)
