Der Graph einer ganzrationalen Funktion 3. Grades 
schneidet die x-Achse an den Stellen 0 und -3 und 
hat bei (3|-6) ein Minimum. 
Wie lautet seine Funktionsgleichung?



Allgemeine Form einer ganzrationalen Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b 

4 Bedingungen: 

1. Schneidet die x-Achse an der Stelle 0 bedeutet: 
 
f(0) = 0 --> 

a * 03 + b * 02 + c * 0 + d = 0 --> 

d = 0

(d  = 0 eingesetzt) 
 
f(-3) = 0 --> 

a * (-3)3 + b * (-3)2 + c * (-3) = 0 -->  

 -27a + 9b - 3c = 0      I

(d = 0 eingesetzt) 
 
f(3) = -6 --> 

a * 3³ + b * 32 + c * 3 = -6 -->  
 
27a + 9b + 3c = -6      II
