Der Graph einer ganzrationalen Funktion 3. Grades 
wechselt bei x = 1 das Vorzeichen, berührt bei 
x = 2 die x-Achse und geht durch den Punkt (3|4). 
Wie lautet seine Funktionsgleichung? 



Allgemeine Form einer ganzrationalen Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b 

4 Bedingungen: 

1. Wechselt bei x = 1 das Vorzeichen bedeutet: 

f(1) = 0 --> 

a * 13 + b * 12 + c * 1 + d = 0 --> 

a + b + c + d = 0       I

2 Berührt bei x = 2 die x-Achse bedeutet zum einen: 

f(2) = 0 --> 

a * 23 + b * 22 + c * 2 + d = 0 -->  

8a + 4b + 2c + d = 0     II 

3. Berührt bei x = 2 die x-Achse bedeutet zum anderen: 

f’(1) = 0 --> 

3a * 22 + 2b * 2 + c = 0 --> 

12a + 4b + c  = 0       III 

4. Geht durch den Punkt (3|4) bedeutet: 

f(1) = 4 -->
