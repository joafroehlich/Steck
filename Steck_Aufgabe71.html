Allgemeine Form einer ganzrationalen 
Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b

zum einen: Steigung der Tangente = 11

27a + 6b + c = 11      I

Funktionsgleichung  y = 11x - 27 bedeutet 
zum anderen mit: 

y(3) = f(3) = 11 * 3 - 27 = 6 

f(3) = 6 --> 

a * 33 + b * 32 + c * 3 + d = 6 -->  

27a + 9 b + 3c + d = 6       II 

3. Hat bei (1|0) einen Wendepunkt bedeutet zum 
einen:

6a * 1 + 2b = 0 --> 

6a + 2b = 0      III
