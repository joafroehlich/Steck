Aufgabe 122

Die Kugel eines Kugelstoßers beschreibt eine 
parabelförmige Bahn wie dargestellt. 
Welche maximale Höhe erreicht die Kugel?



3 Bedingungen: 

1. Geht durch den Punkt (0|1,5) bedeutet: 

f(0) = 1,5 --> 

a * 02 + b * 0 + c = 1,5 --> 

c = 1,5  

2. Geht durch den Punkt (19|0) bedeutet: 
(c = 1,5 eingesetzt) 

f(19) = 0 --> 

361a + 19b = -1,5         I

3. Hat an der Stelle x = 19 den Steigungswinkel 30° bedeutet: 

Steigungswinkel 30° entspricht einer Steigung von 

tan (150)° = - 0,577, 

(mathematische Drehrichtung entgegen dem Uhrzeigersinn) 

f’(19) = -0,577 --> 

2a * 19 + b = -0,577 --> 

38a + b = -0,577      II

I + II * (-19) 

 361a + 19b = -1,5 
-722a - 19b = 10,96
--------------------
-361a       = 9,46 |:361 

          a = -0,026 
 
a = -0,026 in I eingesetzt: 
 
361 * (- 0,026) + 19b = -1,5 
 
         - 9,46 + 19b = -1,5 |+9,46 
 
                  19b = 7,96 |:19 
 
                    b = 0,419 
       
f(x) = -0,026x2 + 0,419x + 1,5 
 
Maximale Höhe: 
 
f’(x) = 0 

 -0,052x + 0,419 = 0 |-0,419 
 
         -0,052x = -0,419 |:0,052 
 
               x = 8,06 m 
 
Die maximale Höhe ist 8,06 m entfernt und beträgt: 
 
f(8,06) = -0,026 * 8,062 + 0,419 * 8,06 + 1,5 = 0   
 
    = -1,689 + 3,377 + 1,5 = 3,188 -->
