Ein Produzent hat Grenzkosten von 38 GE bei 2 ME, 
Stückkosten von  42 GE bei 4 ME, Gesamtkosten von 
504 GE bei 12 ME, und Betriebsoptimum und Betriebs-
minimum fallen zusammen. Er stellt diesen Zusammenhang 
durch eine ganzrationale Kostenfunktion dar. Bei 
welcher ME treten Betriebsoptimum und Betriebsminimum auf?       



Betriebsoptimum und Betriebsminimum fallen zusammen bedeutet: 
Stückkostenfunktion = variable Stückkostenfunktion, 
keine Fixkosten 
 
k(x) = kv(x) --> d = 0 
 
                K(x)     Kv(x)     ax3 + bx2 + cx                           
k(x) = kv(x) = ------ = ------- = ----------------- = ax2 + bx + c  
                  x        x              x

Allgemeine Form einer ertragsgesetzlichen Kostenfunktion 3. Grades:

K(x) = ax3 + bx2 + cx + d 

K’(x) = 3ax2 + 2bx + c 

K’’(x) = 6ax + 2b  

3 Bedingungen:

Das Betriebsminimum ist das Minimum der variablen
