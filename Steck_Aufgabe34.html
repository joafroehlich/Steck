Aufgabe 34

Der Graph einer ganzrationalen Funktion 3. Grades 
berührt die x-Achse im Koordinatenursprung. geht 
durch den Punkt (-3|0) und hat dort die 
Steigung 9. 
Wie lautet seine Funktionsgleichung?



Allgemeine Form einer ganzrationalen Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b 

4 Bedingungen: 

1. Berührt die x-Achse im Koordinatenursprung bedeutet zum einen: 
 
f(0) = 0 --> 

a * 03 + b * 02 + c * 0 + d = 0 --> 

d = 0  
 
2. Berührt die x-Achse im Koordinatenursprung bedeutet zum anderen: 
 
f’(0) = 0 --> 

3a * 02 + 2b * 0 + c = 0 --> 

c = 0      
 
3. Geht durch den Punkt (-3|0) bedeutet: 
(c = 0 und d = 0 eingesetzt) 
  
f(-3) = 0 --> 

a * (-3)3 + b * (-3)2 =  0 --> 

-27a + 9b = 0         I 
 
4. Hat im Punkt (-3|0) die Steigung 9 bedeutet: 
(c = 0 eingesetzt)  
 
f’(-3) = 0 --> 

3a * (-3)2 + 2 b * (-3) --> 

27a - 6b = 9           II 
 
I + II 
 
-27a + 9b = 0 
 27a - 6b = 9 
 --------------
       3b = 9 |:3 
 
        b = 3    
 
b = 3 in II eingesetzt: 
 
27a - 6 * 3 = 9 
 
27a - 18 = 9 |+18 
         
     27a = 27 |:27 
 
       a = 1 

Gesuchte Funktionsgleichung: 

f(x) = x3 + 3x2
