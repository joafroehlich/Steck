ganzrationalen Funktion 5. Grades hat in (-1|1) 
einen Wendepunkt mit der Steigung 3. 
Wie lautet seine Funktionsgleichung?



Allgemeine Form einer zu (0|0) punktsymmetrischen 
ganzrationalen Funktion 5. Grades: 
 
f(x) = ax5 + cx3 + ex 
 
f(x) = 5ax4 + 3cx4 + e 
 
f’’(x) = 20ax3 + 6cx 

4 Bedingungen : (eine mehr als nötig) 
 
1. Geht durch (0|0) bedeutet: 
 
f(0) = 0 -->

0 = 0 liefert kein Ergebnis

einen: 
 
f(-1) = 1 --> 

a * (-1)5 + c * (-1)3 + e * (-1) = 1 --> 

 -a - c - e = 1            I 
 
3. Hat in (-1|1) einen Wendepunkt bedeutet zum 
anderen: 
 
f’’(-1) = 0 --> 

20a * (-1)3 + 6c * (-1) = 0 --> 

-20a - 6c = 0          II  
 
4. Hat in (-1|1) einen Wendepunkt mit der 
Steigung 3 bedeutet: 
 
f’(-1) = 3 --> 

5a * (-1)4 + 3c * (-1)2 + e = 3 --> 

5a + 3c + e = 3       III
