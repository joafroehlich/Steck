Allgemeine Form einer ganzrationalen 
Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b

3a + 2b + c = 0      II

27a + 9 b + 3c + d = 2       III
