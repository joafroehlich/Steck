Der Graph einer ganzrationalen Funktion 3. Grades 
schneidet die x-Achse bei -1,5, hat bei (-0,5|4,5) 
einen Hochpunkt und an der Stelle x = 2/3 einen 
Wendepunkt. Wie lautet seine Funktionsgleichung?



Allgemeine Form einer ganzrationalen 
Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b 

4 Bedingungen: 

1. Schneidet die x-Achse bei -1,5 bedeutet: 
 
f(-1,5) = 0 --> 

a * (-1,5)3 + b * (-1,5)2 + c * (-1,5) + d = 0 -->  

 -3,375a + 2,25b - 1,5c + d = 0         I 
 
2. Hat bei (-0,5|4,5) einen Hochpunkt 
bedeutet zum einen: 
 
f(-0,5) = 4,5 --> 

a * (-0,5)3 + b * (-0,5)2 + c * (-0,5) + d = 0 -->

 - 0,125a + 0,25b - 0,5c + d = 4,5       II  
 
3. Hat bei (-0,5|4,5) einen Hochpunkt 
bedeutet zum anderen: 
 
f’(-0,5) = 0 --> 

3a * (-0,5)2 + 2b * (-0,5) + c = 0 -->  
 
0,75a - b + c = 0       III  
 
4. Hat bei x = 2/3 einen Wendepunkt bedeutet: 
 
f’’(2/3) = 0 --> 

6a * 2/3 + 2b = 0 --> 

4a + 2b = 0       IV 
 
II * (-1) + I

a = 1 in IV eingesetzt:

    4 + 2b = 0 |-4

a = 1 und b = -2 in III eingesetzt.
