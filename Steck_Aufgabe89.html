Allgemeine Form einer achsensymmetrischen ganzrationalen 
Funktion 4. Grades: 
 
f(x) = ax4 + cx2 + e 
 
f’(x) = 4ax3 + 2cx  
 
f’’(x) = 12ax2 + 2c  

3 Bedingungen: 

1. Hat einen Wendepunkt im Punkt (3|31,5) bedeutet zum einen:  
 
f(3) = 31,5 --> 

a * 34 + c * 32 + e = 31,5 --> 

81a + 9c + e = 31,5       I 
 
2. Hat einen Wendepunkt im Punkt (3|31,5) bedeutet zum anderen:  
 
f’’(3) = 0 --> 

12a * 32 + 2c = 0 --> 

108a + 2c = 0     II 
 
3. Hat einen Wendepunkt im Punkt (3|31,5) mit der 
Steigung -18 bedeutet:  
 
f(3) = -18 --> 

4a * 33 + 2c * 3 = - 18 --> 

108a + 6c = -18       III 
 
II * (-1) + III 

-108a - 2c =  0 
 108a + 6c = -18 
-----------------
        4c = -18 |:4 
 
         c = -4,5  
 
c = -4,5 in II eingesetzt: 
 
108a + 2 * (-4,5) = 0 
 
         108a - 9 = 0 |+9 
 
             108a = 9 | :108 
 
                       9      1   
                 a = ----- = ---- 
                      108     12 
 
a = 1/12 und c  = -4,5 in I eingesetzt: 
 
81 * (1/12) + 9 * (-4,5) + e = 31,5 
 
             6,75 - 40,5 + e = 31,5 
 
                  -33,75 + e = 31,5 |+33,75

f(x) = (1/12)x4 - 4,5x2 + 65,25
