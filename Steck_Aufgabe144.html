Aufgabe 144

In einer forstwirtschaftlichen Untersuchung ermittelt 
man wochenweise den Befall von Schadstoffinsekten und 
stellt ihn mit Hilfe einer ganzrationalen Funktion 
3. Grades dar. Die Messung beginnt nach 0,4 Wochen noch 
ohne einen Befall. Nach 2 Wochen beträgt er 48, nach 
3,4 Wochen 81,6 und nach 4 Wochen 72. 
Nach wieviel Wochen ist er wieder verschwunden? 



Allgemeine Form einer ganzrationalen Funktion 3. Grades: 

f(x) = ax3 + bx2 + cx + d 

f’(x) = 3ax2 + 2bx + c 

f’’(x) = 6ax + 2b  

4 Bedingungen: 

1. Die Messung beginnt nach 0,4 Wochen noch ohne 
einen Befall bedeutet: 

f(0,4) = 0 --> 

a * 0,43 + b * 0,42 + c * 0,4 + d = 0 -->  

0,064a + 0,16b + 0,4c + d = 0         I 

2. Nach 2 Wochen beträgt er 48 bedeutet:  

f(2) = 48 --> 

a * 23 + b * 22 + c * 2 + d = 48 --> 

8a + 4b + 2c + d = 48        II 

3. Nach 3,4 Wochen beträgt er 81,6 bedeutet:

39,304a + 11,56b + 3,4c + d = 81,6     III

Gesuchte Funktion: 
 
f(x) = -5x3 + 27x2 - 10x 
 
Bedingung für die Nullstellen: 
 
f(x) = 0

 -x(5x2 - 27x + 10) = 0  --> 
 
 x1 = 0 
 
Eine zweite Nullstelle liegt bei x2 = 0,4 
(siehe Aufgabenstellung)
