Der Graph einer ganzrationalen Funktion 3. Grades 
geht durch den Koordinatenursprung, schneidet bei 
x = 6 die x-Achse, und die Wendetangente durch 
(0|0) hat die Gleichung y = 2x. 
Wie lautet seine Funktionsgleichung?



Allgemeine Form einer ganzrationalen 
Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b 

4 Bedingungen: 

1. Geht duch den Koordinatenursprung bedeutet: 
 
f(0) = 0 --> 

a * 03 + b * 02 + c * 0 + d = 0 --> 

d = 0 
 
2. Schneidet bei x = 6 die x-Achse bedeutet: 
(d = 0 eingesetzt)
 
f(0) = 0 --> 

a * 63 + b * 62 + c * 6 = 0 --> 

216a + 36b + 6c = 0      I  
 
3. Die Wendetangente durch (0|0) bedeutet, (0|0) 
ist Wendepunkt: 
 
f’’(0) = 0 --> 

6a * 0 + 2b = 0 --> 

2b = 0 |:2 --> 

b = 0  
 
4. Wendetangente hat die Gleichung y = 2x 
bedeutet:(b = 0 eingesetzt) 
 
f’(0) = 2 --> 

3a * 02 + 2b * 0 + c  = 2 --> 

c = 2        
 
c = 2 und b = 0 in I eingesetzt: 
 
216a + 6 * 2 = 0 
 
216a + 12 = 0 |-12 
 
     216a = -12 |:216

Gesuchte Funktionsgleichung: 
 
f(x) = -(1/18)x3 + 2x
