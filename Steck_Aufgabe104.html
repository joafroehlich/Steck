Aufgabe 104

Der Graph einer ganzrationalen Funktion 4. Grades 
schneidet die y-Achse bei 4, berührt die x-Achse 
bei x = 2 und hat im Punkt (1|2,25) die Steigung -3. 
Wie lautet seine Funktionsgleichung?



Allgemeine Form einer ganzrationalen Funktion 4. Grades: 

f(x) = ax4 + bx3 + cx2 + dx + e 

f’(x) = 4ax3 + 3bx2 + 2cx + d 

f’’(x) = 12ax2 + 6bx + 2c  

5 Bedingungen:

1. Schneidet die y-Achse bei 4 bedeutet:

f(0) = 4 --> 

a * 04 + b * 03 + c * 02 + d * 0 + e = 4 --> 

e = 4 

2. Berührt die x-Achse bei x = 2 bedeutet zum einen: 
(e = 4 eingesetzt) 

f(2) = 0 --> 

a * 24 + b * 23 + c * 22 + d * 2 + 4 = 0 -->  

16a + 8b + 4c + 2d + 4 = 0         I 

3. Berührt die x-Achse bei x = 2 bedeutet zum anderen: 

f’(2) = 0 --> 

4a * 23 + 3b * 22 + 2c * 2 + d = 0 --> 

32a + 12b + 4c + d = 0            II 

4. Hat im Punkt (1|2,25) die Steigung -3 bedeutet zum einen: 
(e = 4 eingesetzt) 

f(1) = 2,25 --> 

a * 14 + b * 13 + c * 12 + d * 1 + 4 = 2,25 -->  

a + b + c + d + 4 = 2,25       III 

5. Hat im Punkt (1|2,25) die Steigung -3 bedeutet zum anderen: 

f’(1) = -3 --> 

4a * 13 + 3b * 12 + 2c * 1 + d = -3 --> 

4a + 3b + 2c + d = -3      IV
