Aufgabe 145

Hochspannungsleitungen hängen zwischen den gleich 
hohen Masten durch. Höhenmessungen hatten 
folgende Ergebnisse: 20 m von einem Masten 
entfernt hängt die Leitung in 42,8 m Höhe, im 
Abstand von 80 m sind es 35 m und in einem von 
120 m sind es 31,8 m. Wie weit ist der tiefste 
Punkt entfernt? 



Es sind 3 Punkte angegeben, deswegen ist von 
einer ganzrationalen Funktion 2. Grades 
auszugehen.

Allgemeine Form einer ganzrationalen 
Funktion 2. Grades: 

f(x) = ax2 + bx + c 

f’(x) = 2ax + b

f’’(x) = 2a  

3 Bedingungen: 

1. 20 m von einem Masten entfernt hängt die 
Leitung in 42,8 m Höhe bedeutet: 

f(20) = 42,8 --> 

a * 202 + b * 20 + c = 42,8 -->  

400a + 20b + c = 42,8         I 

2. Im Abstand von 80 m sind es 35 m bedeutet:  

f(80) = 35 --> 

a * 802 + b * 80 + c = 35 -->  

6400a + 80b + c = 35        II 

3. In einem von 120 m sind es 31 m bedeutet:  

f(120) = 31,8 --> 

a * 1202 + b * 120 + c = 31.8 -->  

14400a + 120b + c = 31,8     III

I * (-1) + II 

-400a - 20b - c = -42,8 
6400a + 80b + c =  35
------------------------
6000a + 60b     = -7,8       IV   

I * (-1) + III 

 -400a -  20b - c = -42,8 
14400a + 120b + c =  31,8 
--------------------------
14000a + 100b     = -11            V 
 
IV * (-10) + V * 6 

-60000a - 600b  =  78 
 84000a + 600b  = -66 
----------------------
 24000a         = 12 |:24000 
 
              a = 0,0005    
 
a = 0,0005 in IV eingesetzt: 
 
6000 * 0,0005 + 60b = -7,8 
 
            3 + 60b = -7,8 |-3 
 
                60b = -10,8 |:60 
 
                  b = -0,18      
 
a = 0,0005 und b = -0,18 in I eingesetzt: 
 
400 * 0,0005 + 20 * (-0,18) + c = 42,8 
 
                  0,2 - 3,6 + c = 42,8 

                       -3,4 + c = 42,8  |+3,4 
 
                              c = 46,2 
 
Gesuchte Funktion: 
 
f(x) = 0,0005x2 - 0,18x + 46,2 
 
Wenn die Masten gleich hoch sind, dann ist der 
tiefste Punkt der Scheitelpunkt der Parabel: 
 
Scheitelpunktform: 
 
f(x) = 0,0005x2 - 0,18x + 46,2 
 
f(x) = 0,0005(x2 - 360x + 92400) 
 
Quadratische Ergänzung: 
 
f(x) = 0,0005(x2 - 360x + 32400 - 32400 + 92400) 

f(x) = 0,0005((x - 180)2 + 60000) 

Ausmultiplizieren: 

f(x) = 0,0005(x - 180)2 + 30 

Der tiefste Punkt der Leitung hat einen 
Abstand von 180 m und eine Höhe von 30 m. 

Alternative Rechnung: 

Minimum bestimmen: 

Bedingung f’(x) = 0 

0,001x - 0,18 = 0 |+0,18 

       0,001x = 0,18 |:0,001 

            x = 180 m
