Der Graph einer ganzrationalen Funktion 2. Grades 
geht durch die Punkte (-1|2), (1|3) und (3|2). 
Wie lautet seine Funktionsgleichung? 


 
Allgemeine Form einer ganzrationalen 
Funktion 2. Grades: 
  
f(x) = ax² + bx + c 
 
f’(x) = 2ax + b 
 
f’’(x) = 2a 
 
3 Bedingungen: 

1. Geht durch den Punkt (-1|2) bedeutet: 
 
f(-1) = 2 --> 

a * (-1)² + b * (-1) + c = 2 --> 

a - b + c = 2       I 
 
2. Geht durch den Punkt (1|3) bedeutet:  
 
f(1) = 3 --> 

a * 1² + b * 1 + c = 3 --> 

a + b + c = 3            II         
 
3. Geht durch den Punkt (3|2) bedeutet:  
 
f(3) = 2 --> 

a * 3² + b * 3 + c = 2 --> 

9a + 3b + c = 2         III 
 
I + II

---------------

------------------
12a +      4c = 8      V
