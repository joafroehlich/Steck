Der Graph einer achsensymmetrischen ganzrationalen 
Funktion 4. Grades geht durch den Koordinaten-
ursprung und schneidet die x-Achse an der Stelle 
x = 3 mit der Steigung -48. 
Wie lautet seine Funktionsgleichung? 



Allgemeine Form einer achsensymmetrischen 
ganzrationalen Funktion 4. Grades: 
 
f(x) = ax4 + cx2 + e 
 
f’(x) = 4ax3 + 2cx  
 
f’’(x) = 12ax2 + 2c  

3 Bedingungen: 

1. Geht durch den Koordinatenursprung bedeutet: 
 
f(0) = 0 --> 

a * 04 + c * 02 + e = 0 --> 

e = 0           
 
2. Schneidet die x-Achse an der Stelle x = 3 
bedeutet: (e = 0 eingesetzt) 
 
f(3) = 0 --> 

a * 34 + c * 32 = 0 --> 

81a + 9c = 0          I 
 
3. Schneidet die x-Achse an der Stelle x = 3 mit 
der Steigung -48 bedeutet:  
 
f’(3) = - 48 --> 

4a * 33 + 2c * 3 = - 48 --> 

108a + 6c = - 48         II 
 
I * (-6) + II * 9 

-486a - 54c =  0 
 972a + 54c = -432
