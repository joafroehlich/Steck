Allgemeine Form einer ganzrationalen 
Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b 

4 Bedingungen: 

1. Geht durch den Nullpunkt bedeutet:

d = 0

zum einen: (d = 0 eingesetzt):

a * 13 + b * 12 + c * 1 = -2 --> 

a + b + c = -2         I      
 
3. Hat einen Wendepunkt bei (1|-2) bedeutet 
zum anderen: 
 
f’’(1) = 0 --> 

6a * 12 + 2b = 0 --> 

6a + 2b = 0              II

(2|0) bedeutet:
