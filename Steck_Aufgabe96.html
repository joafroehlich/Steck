Allgemeine Form einer ganzrationalen 
Funktion 4. Grades: 

f(x) = ax4 + bx3 + cx2 + dx + e 

f’(x) = 4ax3 + 3bx2 + 2cx + d 

f’’(x) = 12ax2 + 6bx + 2c  

5 Bedingungen: 

1. Berührt an der Stelle x = -1 die x-Achse 
bedeutet zum einen:

f(-1) = 0 --> 

a * (-1)4 + b * (-1)3 + c * (-1)2 + d * (-1) + e = 0 -->  

a - b + c - d + e = 0        I 

2. Berührt an der Stelle x = -1 die x-Achse 
bedeutet zum anderen: 

f’(-1) = 0 --> 

4a * (-1)3 + 3b * (-1)2 + 2c * (-1) + d = 0 --> 

-4a + 3b - 2c + d = 0       II 

3. Hat im Punkt (2|6,75) einen Sattelpunkt 
bedeutet erstens: 

f(2) = 6,75 --> 

a * 24 + b * 23 + c * 22 + d * 2 + e = 6,75 -->  

16a + 8b + 4c + 2d + e = 6,75              III 

4. Hat im Punkt (2|6,75) einen Sattelpunkt 
bedeutet zweitens:  

f’(2) = 0 --> 

4a * 23 + 3b * 22 + 2c * 2 + d = 0 -->  

32a + 12b + 4c + d = 0        IV
