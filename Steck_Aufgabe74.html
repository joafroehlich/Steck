hat ein Maximum bei (2|4) und ein Minimum bei 
(1|1). Wie lautet seine Funktionsgleichung? 



Allgemeine Form einer ganzrationalen 
Funktion 3. Grades: 
 
f(x) = ax3 + bx2 + cx + d 
 
f’(x) = 3ax2 + 2bx + c 
 
f’’(x) = 6ax + 2b

12a + 4b + c = 0       II

f(1) = 1 --> 

a * 13 + b * 12 + c * 1 + d = 1 --> 

a + b + c + d = 1     III 

4. Hat ein Minimum bei (1|1) bedeutet zum anderen: 

f’(1) = 0 --> 

3a * 12 + 2b * 1 + c = 0 --> 

3a + 2b + c = 0       IV
