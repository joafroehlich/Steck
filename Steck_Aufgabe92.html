Aufgabe 92

Der Graph einer ganzrationalen Funktion 4. Grades 
hat im Punkt (0|0) die Wendetangente y = x und im 
Punkt (2|4) die Steigung 0. 
Wie lautet seine Funktionsgleichung?



Allgemeine Form einer ganzrationalen Funktion 4. Grades: 

f(x) = ax4 + bx3 + cx2 + dx + e 

f’(x) = 4ax3 + 3bx2 + 2cx + d 

f’’(x) = 12ax2 + 6bx + 2c  

5 Bedingungen: 

1. Hat im Punkt (0|0), dem Wendepunkt, bedeutet zum einen:  
 
f(0) = 0 --> 

a * 04 + b * 03 + c * 02 + d * 0 + e = 0 --> 

e = 0  
 
2. Hat im Punkt (0|0), dem Wendepunkt, bedeutet zum anderen:  
 
f’’(0) = 0 --> 

12a * 02 + 6b * 0 + 2c = 0 --> 

2c = 0 |:2 --> 

c = 0 
 
3. Hat die Wendetangente y = x bedeutet: 
(Steigung = 1) 
 
f'(0) = 1 --> 

4a * 03 + 3b * 02 + 2c * 0 + d = 1 --> 

d = 1 
 
4. Hat im Punkt (2|4) die Steigung 0 bedeutet zum einen: 
(c = 0 und  d = 1 und e = 0 eingesetzt)  
 
f(2) = 4 --> 

a * 24 + b * 23 + 2 = 4 --> 

16a + 8b + 2 = 4       I 
 
5. Hat im Punkt (2|4) die Steigung 0 bedeutet zum anderen: 
(c = 0 und  d = 1 eingesetzt)  
 
f’(2) = 0 --> 

4a * 23 + 3b * 22 + 1 = 0 --> 

32a + 12b + 1 = 0          II 
 
I * (-2) + II 

-32a - 16b - 4 = -8 
 32a + 12b + 1 = 0 
--------------------
       -4b - 3 = -8 |+3 
 
           -4b = -5 |:(-4) 
 
             b = 5/4 = 1,25        
 
b = 1,25 in I eingesetzt: 
 
16a + 8 * 1,25 + 2 = 4 
 
      16a + 10 + 2 = 4 
 
          16a + 12 = 4 |-12 
 
               16a = -8 |:16 
 
                 a = -8/16 = -0,5 
 
Gesuchte Funktionsgleichung: 
 
f(x) = -0,5x4 + 1,25x3 + x
